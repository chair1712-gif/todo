# Todo App with API

Aplikasi Todo List lengkap dengan backend API menggunakan Netlify Functions.

## Fitur
- ✅ CRUD lengkap (Create, Read, Update, Delete)
- ✅ Filter (Semua/Aktif/Selesai)
- ✅ Statistik real-time
- ✅ Responsive design
- ✅ API documentation in-app
- ✅ Status monitoring

## Teknologi
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Netlify Functions (Serverless)
- **Hosting**: Netlify (Frontend + Backend)

## Cara Menjalankan

### 1. Lokal Development
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Clone repository
git clone [repo-url]
cd spatra-todo-app

# Jalankan lokal
netlify dev
